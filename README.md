# 🚗 Vehicle Vault

A web application to manage and compare vehicles with smart features.

---

## 🔥 Features

- 🔐 User Authentication (Login/Register)
- 🚗 Add / Edit / Delete Vehicles
- ⚔️ Compare Vehicles
- 🏆 Smart Winner Highlight System
- 🎨 Clean UI (Card Based Design)

---

## 🛠️ Tech Stack

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server (LocalDB)
- HTML, CSS
