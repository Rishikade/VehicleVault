# 🚗 Vehicle Vault

A web application to manage and compare vehicles with smart features.

---

## 🔥 Features

- 🔐 User Authentication (Login/Register)
- 🚗 Add / Edit / Delete Vehicles
- ⚔️ Compare Vehicles
- 🏆 Smart Winner Highlight System
- 🎨 Clean UI (Card Based Design)

---

## 🛠️ Tech Stack

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server (LocalDB)
- HTML, CSS

---

---

## 🚀 How to Run

1. Clone repo
2. Open in Visual Studio
3. Run migrations:# VehicleVault
4. Run project

---

## 💡 Future Improvements

- Image upload feature
- Better UI animations
- AI-based recommendations


---
## Architecture

- Frontend: Razor Views (ASP.NET MVC)
- Backend: ASP.NET Core Controllers + Entity Framework
